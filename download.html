The all new browser

WBINARY allows you to watch videos ten times faster without any cookie catching.

With your old browser Youtube or anything else can get existing data of what you are watching.

WBinary does not save cookies or search history, so Youtube cannot get existing data.

Steps to install:

1. Install Python from the Microsoft store

2. Install all libraries using pip

3. Run WBINARY using "python C:Users{Your Username}Downloadswbinary-beta.py"

At step 3, do not use {Your Username}, instead use your actual Windows Username
